Using Python3

Install python :
    $ brew install python
      
Check the version using the command:
    $ python3
    
On executing getReponse.py, the latitudes and longitudes of the given locations are stored in the form of list of dictionaries in outfile so that it need not be done again and again to save a lot of time.

Finally on executing ./instazen.py , the user enters the location and lists of the location closer to the users location.
